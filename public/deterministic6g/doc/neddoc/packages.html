Name Type Description
UdpEdgeCloudApp simple module

class UdpEdgeCloudApp The UdpEdgeCloudApp module is designed to simulate a UDP-based application in edge cloud scenarios. This module builds upon the UdpEchoApp, inheriting its functionalities and adding the capability to introduce specific delays. Such delays might be used to simulate various network conditions or service latencies in edge cloud settings. Users can specify destination addresses, destination ports, and custom delays to simulate application behavior in edge environments.

PairwiseDelayer simple module

class PairwiseDelayer The PairwiseDelayer module is designed to delay packets based on pairwise configurations. This module works by accepting packets and applying a delay according to the specified delayConfig. This can be particularly useful in scenarios where varying delays are required for different packet pairs or flows. It's an extension of PacketDelayerBase and adheres to the IPacketDelayer contract. Users can provide custom delay configurations using the delayConfig XML parameter.
