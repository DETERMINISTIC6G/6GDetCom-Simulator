AGV compound module

The module extends UnitWithAdapter to represent an AGV in a cooperative manufacturing environment. It has the ability to communicate wirelessly and has two Ethernet submodules: one for communication between AGVs (interAgv) and another for communication to a processing cell (interToCell).
ProcessingCell compound module

The module is designed to simulate a processing unit in a cooperative manufacturing environment. It extends UnitWithAdapter, adding specific configurations like Ethernet bitrate and submodules for controlling swarm operations and status.

Source code

//
// This module provides a network communication scenarios based on whether the communication is wireless.
// It configures the necessary submodules ,connections,and gates to facilitate dynamic communication pattern.
//
module UnitWithAdapter
{
    parameters:
        bool isWireless = default(true); //indicates whether the scenario is wireless
    gates:
        inout adapterGate; //interface for adapter's functionality
    submodules:
        detComAdapter: DetComAdapter {
            @display("p=350,200");
        }
    connections:
        if (isWireless) {
            detComAdapter.ueg++ <--> { @display("ls=,,da"); } <--> adapterGate;
        }
        if (!isWireless) {
            detComAdapter.ethg++ <--> { @display("ls=,,da"); } <--> adapterGate;
        }
}
