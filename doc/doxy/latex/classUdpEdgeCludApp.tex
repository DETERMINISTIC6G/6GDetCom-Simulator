\doxysubsection{Detailed Description}
The Udp\+Edge\+Cloud\+App module is designed to simulate a UDP-\/based application in edge cloud scenarios. 

This module builds upon the Udp\+Echo\+App, inheriting its functionalities and adding the capability to introduce specific delays. Such delays might be used to simulate various network conditions or service latencies in edge cloud settings. Users can specify destination addresses, destination ports, and custom delays to simulate application behavior in edge environments.
