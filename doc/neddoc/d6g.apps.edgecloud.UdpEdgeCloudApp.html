UdpEdgeCloudApp

Package: d6g.apps.edgecloud

UdpEdgeCloudApp

simple module

C++ definition

The module is designed to simulate a UDP-based application in edge cloud scenarios. This module builds upon the UdpEchoApp, inheriting its functionalities and adding the capability to introduce specific delays. Such delays might be used to simulate various network conditions or service latencies in edge cloud settings. Users can specify destination addresses, destination ports, and custom delays to simulate application behavior in edge environments.

// The module is designed to simulate a UDP-based application in edge cloud scenarios.
// This module builds upon the UdpEchoApp, inheriting its functionalities and adding the capability

simple UdpEdgeCloudApp extends UdpEchoApp
{
    parameters:
        @class(UdpEdgeCloudApp);

        string destAddress = default(""); // destination IP address for the UDP packets

        int destPort; // destination port number for the UDP packets

        volatile double delay @unit(s) = default(0s); // specifies delay per UDP packets
}
